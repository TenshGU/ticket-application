# ticket-application
The System of ticket（飞机订票系统）——2021.7.25开源，以此纪念我和猴哥和+锐的第一个前后端分离+外网访问项目
React+SpringBoot+MyBatis+SpringSecurity+MySql+Redis
Linux+Nginx(阿里云服务器部署，外网访问)
更多细节点：AOP、IP-验证码、布隆过滤器
